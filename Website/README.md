# Better Call Wes - Website

This is the static HTML/CSS website for Better Call Wes Plumbing & Heating.

## Structure

- `index.html`: Homepage
- `about.html`: About page
- `contact.html`: Contact/Booking page
- `pricing.html`: Full price list
- `reviews.html`: Testimonials
- `services/`: Specific service pages (Boilers, Repairs, etc.)
- `css/`: Stylesheet (Design System)
- `js/`: Main JavaScript file
- `assets/`: Images and icons

## Editing

To make changes, simply edit the HTML files directly.
The CSS is in `css/styles.css` and uses CSS variables for colors/fonts at the top of the file.

### colors
- Primary: Navy `#0f2942`
- Accent: Amber `#e8a435`

## Launching

You can deploy this to any static host (Netlify, Vercel, GitHub Pages) or simply upload the files to a cPanel hosting account.
